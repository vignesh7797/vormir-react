- `apps/storybook/stories/Timeline.stories.tsx` (7 errors)
- `apps/storybook/stories/Stat.stories.tsx` (6 errors)

**Issues:**
- Story type definitions missing required `args` property
- Unused imports (React, toast, ChevronRight, etc.)
- Timeline stories missing required `children` property

**Solution:**
```typescript
// Before
export default {
  title: 'Components/Toast',
  component: Toast,
} as Meta<typeof Toast>;

// After
export default {
  title: 'Components/Toast',
  component: Toast,
  args: {}, // Add required args property
} as Meta<typeof Toast>;
```
