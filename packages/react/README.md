# @vormir/react

A modern, customizable React UI library with built-in light/dark mode support.

## Installation

```bash
npm install @vormir/react
# or
pnpm add @vormir/react
# or
yarn add @vormir/react
```

## Usage

```tsx
import { ThemeProvider, Button } from '@vormir/react';

function App() {
  return (
    <ThemeProvider>
      <Button variant="solid" colorScheme="brand">
        Click me
      </Button>
    </ThemeProvider>
  );
}
```
