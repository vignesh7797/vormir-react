## Getting Started

### Installation

```bash
npm install @vormir/react
```

### Basic Usage

```tsx
import { ThemeProvider, Button } from '@vormir/react';

function App() {
  return (
    <ThemeProvider>
      <Button>Click me</Button>
    </ThemeProvider>
  );
}
```
